# ESLint reports a vue/require-default-prop error for certain vue-types validators.

If your uses eslint-plugin-vue v6 or lower, you might receive a linting error related to the vue/require-default-prop (opens new window) rule for validators using the .isRequired or .loose flag (see vue-types#179 (opens new window)).

The best solution to fix this issue is to upgrade the library eslint-plugin-vue to v7.

However, if you cannot upgrade the library, you can install eslint-plugin-vue-types (opens new window). The plugin extends eslint-plugin-vue and filters out most incorrectly flagged usage cases.

Read the plugin documentation (opens new window) for installation and usage details.
